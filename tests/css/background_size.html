fixes191b — background-size

Tile-size resolution for bitmap backgrounds. Unset preserves the historical MacSurf default (single tile = box).

BS1 — no background-size (default)

Current default behaviour. Single tile, may scale to swatch dimensions.

BS2 — background-size: 32px 32px (natural)

Tiles at 32x32 = 10 columns by 5 rows in a 320x160 swatch.

BS3 — background-size: 16px 16px (half)

Tiles at 16x16 = 20 columns by 10 rows. Same icon, scaled down per tile.

BS4 — background-size: contain

Single tile, scaled so the whole image fits inside swatch with aspect preserved. Square image, 160-tall swatch -> 160x160 tile.

BS5 — background-size: cover

Single tile, scaled to cover swatch. Square image, 320-wide swatch -> 320x320 tile (may overflow vertically; clipped to swatch box).

BS6 — background-size: 48px auto (aspect-preserved)

w=48px, h derived from natural aspect ratio (32x32 square -> 48x48). Tiles at 48x48.
