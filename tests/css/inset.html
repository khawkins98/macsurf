fixes191a — inset shorthand

Each ::before/overlay is positioned via the inset shorthand. The preprocessor expands to top/right/bottom/left so libcss handles the longhands normally.

I1 — inset: 0

Overlay should fill the entire card (all four edges flush).

I2 — inset: 8px

Overlay inset 8px from every edge.

I3 — inset: 4px 20px (V/H)

4px top/bottom, 20px left/right.

I4 — inset: 4px 20px 16px (T R+L B)

top=4, right=20, bottom=16, left=20.

I5 — inset: 4px 20px 16px 60px (TRBL)

top=4, right=20, bottom=16, left=60 — visibly asymmetric.
