live in [CLAUDE.md](../CLAUDE.md) under "Build State"; the full text of every shipped fix is here.

Entries are in reverse chronological order (newest first). Each entry describes
the root cause and the shape of the fix — useful when a new bug looks like a
re-emergence of an old one, or when an agent needs to understand why a
